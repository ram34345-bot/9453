# 山海行者（Prototype）

中國神話風 2D 開放世界 RPG 原型，使用 HTML + CSS + JavaScript + Canvas。

## 1) 執行方式
1. 直接用瀏覽器開啟 `index.html`。
2. 建議使用桌機 Chrome / Edge 測試鍵盤操作。
3. 手機開啟時會自動顯示虛擬搖桿與觸控按鈕。

## 2) 怎麼玩（快速）
1. 在青州小鎮找到「鎮長」按 `E` 接主線。
2. 前往右側「妖霧森林」打妖怪（`J` 攻擊）。
3. 擊敗特定妖怪後選擇「斬殺」或「收服」。
4. 收服可獲得妖魂被動：
   - 小燈狐：尋寶提示
   - 山鬼：低血自動護盾
   - 河伯：水系技能冷卻減少

## 3) 操作
### 桌機
- 移動：`WASD` 或方向鍵
- 攻擊：`J`
- 互動：`E`
- 技能：`K`
- 上/下馬：`M`
- 用藥：`B`

### 手機
- 左下虛擬搖桿：移動
- 右下按鈕：攻擊 / 互動 / 技能 / 上馬

## 4) 死亡與存檔
- 死亡後於最近神龕復活，不會重開。
- 保留等級、裝備、任務、背包、妖魂。
- 懲罰：銀兩減少、耐久下降、短暫虛弱。
- 按右上角「存檔 / 讀檔」使用 localStorage 保存進度。
